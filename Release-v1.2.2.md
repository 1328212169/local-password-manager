# 密码管理器 v1.2.2

**发布日期**: 2026-04-28  
**类型**: 功能改进版本

---

##  版本亮点

v1.2.2版本主要优化了悬浮窗口的用户体验，改进了选中状态的视觉效果，让账户密码的管理更加直观和便捷。

---

##  新增功能与改进

###  悬浮窗口优化
- ✅ **优化选中状态显示效果**
  - 选中条目时显示明显的蓝色背景（#e3f2fd）
  - 边框加粗至3px并变为醒目的蓝色（#2196f3）
  - 增强视觉反馈，使选中状态一目了然
  
- ✅ **改进条目样式**
  - 增加未选中条目的边框宽度（从1px到2px）
  - 增大圆角半径（从4px到6px）
  - 提升整体美观度和可读性

### 📋 批量导入优化（v1.2.1）
- ✅ 移除10条预览限制，现在可以查看所有待导入的数据
- ✅ 预览表格支持滚动条，可以上下滑动查看完整列表

---

## 🔧 技术细节

### 修改文件
- `floating_window.py` - 优化悬浮窗口选中状态显示
- `batch_importer.py` - 移除预览数量限制
- `.gitignore` - 添加更严格的文件排除规则

### 依赖
- Python 3.9+
- PyQt6
- Argon2id（密码哈希）
- AES-256-GCM（数据加密）

---

## 📥 下载与安装

### 系统要求
- Windows 7 或更高版本
- 无需安装额外依赖

### 安装步骤
1. 下载 `密码管理器1.2.2.zip`
2. 解压到任意目录
3. 双击 `密码管理器.exe` 运行
4. 首次使用需设置主密码

### 从源码运行
```bash
git clone https://github.com/1328212169/local-password-manager.git
cd local-password-manager
pip install -r requirements.txt
python main.py
```

## ⚠️ 重要提示

### 安全注意事项
1. **主密码**: 请牢记主密码，虽然可以通过邮箱重置，但会丢失历史数据
2. **邮箱绑定**: 强烈建议绑定邮箱用于密码重置
3. **数据备份**: 定期导出数据库到安全位置
4. **本地存储**: 所有数据本地加密存储，不会上传到云端

### 升级说明
- 从v1.2.1升级：直接下载新版本，数据文件兼容
- 从更早版本升级：建议先导出数据库，再导入到新版本
