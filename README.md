# 密码管理器

一个安全、易用的本地密码管理工具，支持多种安全特性和便捷功能。

## 🌟 功能特性

### 🔐 核心安全功能
- **主密码保护**：采用 Argon2id 算法进行密码哈希
- **数据加密**：使用 AES-256-GCM 加密本地存储
- **自动锁定**：支持空闲时间自动锁定
- **手动锁定**：支持手动立即锁定
- **最小化锁定**：可设置最小化到托盘时自动锁定
- **密码生成**：支持生成高强度随机密码
- **剪贴板自动清空**：复制密码后15秒自动清空剪贴板

### 📋 密码管理功能
- **添加/编辑/删除**：完整的密码条目管理
- **批量导入**：支持 CSV/TXT 格式批量操作
- **搜索功能**：快速查找密码条目
- **悬浮窗口**：快捷键 Ctrl+Alt+P 快速访问
- **双击打开网址**：方便快速访问网站

### ⚙️ 系统集成功能
- **系统托盘**：支持最小化到系统托盘
- **全局快捷键**：Ctrl+Alt+P 调用悬浮窗口
- **主题切换**：支持浅色/深色主题
- **操作说明**：详细的使用指南

### 📧 恢复功能
- **邮箱绑定**：支持绑定邮箱用于密码重置
- **验证码验证**：安全的密码重置流程

## 🛠️ 技术栈

- **开发语言**：Python 3.9+
- **GUI框架**：PyQt6
- **加密算法**：
  - 主密码：Argon2id
  - 数据加密：AES-256-GCM
- **密码生成**：Python `secrets` 模块
- **数据存储**：加密 JSON（.json.aes）
- **打包工具**：PyInstaller

## 📦 安装说明

### 方式一：直接运行（推荐）
1. 从 GitHub Releases 下载最新的压缩包
2. 解压到任意目录
3. 双击 `密码管理器.exe` 运行
4. 首次运行需设置主密码

### 方式二：从源码运行
1. 克隆仓库：`git clone <仓库地址>`
2. 安装依赖：`pip install -r requirements.txt`
3. 运行程序：`python main.py`

## 🚀 使用指南

### 1. 首次使用
- 启动程序后，设置一个强主密码
- 建议绑定邮箱，用于密码重置

### 2. 基本操作
- **添加密码**：点击"添加"按钮，填写密码信息
- **编辑密码**：选择条目，点击"编辑"按钮
- **删除密码**：选择条目，点击"删除"按钮
- **搜索密码**：在搜索框中输入关键词

### 3. 悬浮窗口
- 按下 `Ctrl+Alt+P` 打开悬浮窗口
- 可快速搜索和复制密码
- 双击条目可打开对应网址

### 4. 安全设置
- 打开"设置"对话框
- 可调整自动锁定时间
- 可设置最小化时是否锁定
- 可绑定/解绑邮箱

### 5. 批量操作
- **导入**：点击"导入"按钮，选择 CSV/TXT 文件
- **导出**：点击"导出"按钮，选择保存位置

## 🔒 安全说明

1. **数据存储**：所有密码数据本地加密存储，你自己可以备份到你自己的QQ邮箱里面。（备份时请确保邮箱安全）
2. **主密码**：采用 Argon2id 算法哈希，安全性高
3. **加密密钥**：从主密码派生，不直接存储
4. **剪贴板**：密码复制后15秒自动清空
5. **自动锁定**：空闲时间达到设定值后自动锁定
