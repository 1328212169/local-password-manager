## 问题分析

错误信息 `QSystemTrayIcon::setVisible: No Icon set` 表明系统托盘图标没有正确设置。代码中使用 `QIcon.fromTheme("password-manager")` 尝试从系统主题获取图标，但在 Windows 系统上，这个功能可能无法正常工作，或者系统中没有这个图标。

## 解决方案

### 方案1：使用 PyQt6 内置图标
修改 `init_system_tray` 方法，使用 PyQt6 内置的图标替代系统主题图标。

### 方案2：创建自定义图标
创建一个简单的自定义图标文件，并在代码中引用它。

### 方案3：使用 QPixmap 创建基本图标
使用 QPixmap 创建一个简单的基本图标，确保在所有系统上都能正常显示。

## 实施计划

我将选择方案3，因为它不需要外部文件，并且在所有系统上都能可靠工作。

### 修改文件
- **main_window.py**：更新 `init_system_tray` 方法，使用 QPixmap 创建一个简单的图标

### 具体修改
```python
def init_system_tray(self):
    """初始化系统托盘"""
    self.tray_icon = QSystemTrayIcon(self)
    
    # 创建一个简单的图标
    pixmap = QPixmap(32, 32)
    pixmap.fill(Qt.GlobalColor.white)
    painter = QPainter(pixmap)
    painter.setBrush(Qt.GlobalColor.red)
    painter.drawRect(8, 8, 16, 16)
    painter.end()
    
    self.tray_icon.setIcon(QIcon(pixmap))
    
    # 其余代码保持不变
```

## 预期效果

修改后，系统托盘将显示一个简单的红色矩形图标，解决 "QSystemTrayIcon::setVisible: No Icon set" 错误，确保密码管理器能够正常启动和运行。
